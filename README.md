# Projet Déploiement Dolibarr & GLPI

## Description
Script d'automatisation pour le déploiement de Dolibarr et GLPI avec sécurité SSL.

## Installation
```bash
sudo bash deploy_solutions.sh
```

## Accès
- https://dolibarr.local
- https://glpi.local
- http://localhost/ca-cert/ (télécharger certificat CA)
